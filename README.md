## CollaTalk

- 서비스 소개: 협업 소통 앱
- 개발 인원: 1인
- 개발 기간: 24.07.05 ~ **(진행중)**
- 개발 환경
  - 최소버전: iOS 16
  - Portrait Orientation 지원
  - 라이트 모드 지원

<br/>

## 💪 주요 기능

- 회원 인증
  - 회원 가입
  - 로그인 / 소셜 로그인(카카오/애플)
  - 로그아웃
- 프로필 조회 / 편집
- 워크스페이스 기능
  - 워크스페이스 생성 / 조회 / 편집 / 삭제
  - 워크스페이스 관리자 변경
  - 워크스페이스 퇴장
  - 멤버 초대
- 채널 기능
  - 채널 생성 / 조회 / 탐색 / 편집 / 삭제
  - 채널 관리자 변경
  - 채널 퇴장
- 채팅 기능(1:1 채팅 / 단체 채팅)
- 코인 결제 기능

<br/>

## 🛠 기술 소개

- SwiftUI, Swift Concurrency, Combine
- Redux Pattern, Router / Coordinator Pattern, Repository Pattern
- Moya, LocalizedError, Image Caching
- Realm, IAMPort, SocketIO
- KakaoOpenSDK(Kakao Login) / AuthenticationServices(Apple Login)
